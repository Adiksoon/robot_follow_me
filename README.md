Celem projektu jest zaprojektowanie i oprogramowanie robota mobilnego na platformie PiRacer, który autonomicznie podąża za wybranym obiektem wykorzystując kamerę OAK-D Lite w środowisku ROS2. System opiera się na sterowaniu kaskadowym dostosowanym do modelu kinematycznego Ackermanna, wykorzystując sieci neuronowe do detekcji celu oraz symulację w Ignition Gazebo do weryfikacji algorytmów.
